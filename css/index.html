# CSS

# pointer-events

pointer-events CSS 속성은, 특정 엘리먼트의 트리거 역할을 설정합니다. '시각적'으로 트리거 역할(hover, active)을 하는 기능 뿐만 아니라, 이벤트 핸들러가 등록 된 상태의 엘리먼트일지라도 이벤트 트리거로서의 역할을 강제로 막을 수 있습니다.

# Values

  • common
    • auto, none;
  • SVG only
    • visiblePainted;
    • visibleFill;
    • visibleStroke;
    • visible
    • painted;
    • fill;
    • stroke;
    • all;

# 속성

  • counter-reset
    • 값의 초기화
  • counter-increment
    • 값의 증감

# 함수

content 에 사용 가능

  • counter(name, style)
  • counters(name, string, style)

# example code

ex1. counter

body {
  counter-reset: section;
}

h3::before {
  counter-increment: section;
  content: counter(section) '. ';

# 10초 후 링크

10초 후에 링크가 가능해짐. 재미있어 보이나 필요한 이유는 모르겠음.

/* enable click animation */
@keyframes enable-click {
  0% {
    pointer-events: none;
  }
  99% {
    pointer-events: none;
  }
  100% {
    pointer-events: auto;
  }
}

.wrapper {
  display: inline-block;
  background: linear-gradient(to left, transparent 50%, #f60 50%) right;
  background-position: 100% 0%;
  background-size: 200%;
  transition: background 0.2s;
}
.wrapper:hover {
  background-position: left;
  transition: background 10s linear;
}
.wrapper a[href^="item"] {
  pointer-events: none;
}
.wrapper:hover a[href^="item"] {
  animation: enable-click 10.01s forwards;
}

# SVG drop shadow with filter property

SVG에 그림자 효과는 CSS filter 속성으로 처리하면 된다. SVG 테그를 사용할땐 svg 안에 요소가 아닌 svg 요소에 적용해야 한다.

.shadow {
  -webkit-filter: drop-shadow( 3px 3px 2px rgba(0, 0, 0, .7));
  filter: drop-shadow( 3px 3px 2px rgba(0, 0, 0, .7));
}

# CSS fix for 100vh in mobile

height: 100vh 가 모바일에서 잘 작동하지 않을 경우. 대게 사파리인 경우인듯.

html {
  min-height: -webkit-fill-available;
}
body {
  display: flex;
  flex-direction: column;
  margin: 0;
  min-height: 100vh;
  /* mobile viewport bug fix */
  min-height: -webkit-fill-available;
}

main {
  flex: 1;
}
