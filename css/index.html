# CSS

Cascading Style Sheets

# pointer-events

pointer-events CSS 속성은, 특정 엘리먼트의 트리거 역할을 설정합니다. '시각적'으로 트리거 역할(hover, active)을 하는 기능 뿐만 아니라, 이벤트 핸들러가 등록 된 상태의 엘리먼트일지라도 이벤트 트리거로서의 역할을 강제로 막을 수 있습니다.

# Values

  • common
    • auto, none;
  • SVG only
    • visiblePainted;
    • visibleFill;
    • visibleStroke;
    • visible
    • painted;
    • fill;
    • stroke;
    • all;

}
body[data-theme="dark"] {
  --logo: url(logo-light.png);
}

.logo {
  /* ... */

# counter

요소가 몇번 사용되었냐에 따라 값 사용 가능

# 속성

  • counter-reset
    • 값의 초기화
  • counter-increment
    • 값의 증감

# 함수

content 에 사용 가능

  • counter(name, style)
  • counters(name, string, style)

# example code

ex1. counter

body {
  counter-reset: section;
}

h3::before {
  counter-increment: section;
  content: counter(section) '. ';
}

  'Roboto',

# fullscreen mode

전체화면 모드 반응 스타일

@media all and (display-mode: fullscreen) {
  /* apply fullscreen mode */
}
1
2
3

자바스크립트 적용

window.matchMedia('(display-mode: fullscreen)')
  .addListener(({ matches }) => {
    if (matches) {
      // Apply fullscreenmode mode related changes
    } else {
      // Remove fullscreenmode mode related changes
    }
  });

  :root {
    --color-text: #233742;

    --color-bg: white;
  }
}
@media (prefers-color-scheme: dark) {
  :root {
    --color-text: white;
    --color-link: #b4cddd;
    --color-bg: #233742;
  }
}

body {
  color: var(--color-text);
  background-color: var(--color-bg);
}

:root {

  animation-iteration-count: infinite;
}
