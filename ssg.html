# JAMStack

Jamstack is an architecture designed to make the web faster, more secure, and easier to scale. It builds on many of the tools and workflows which developers love, and which bring maximum productivity.

The core principles of pre-rendering, and decoupling, enable sites and applications to be delivered with greater confidence and resilience than ever before.

Jamstack은 웹을 더 빠르고, 더 안전하며, 확장하기 쉽게 하기 위해 고안된 아키텍처다. 개발자가 좋아하고 최대 생산성을 가져다 주는 많은 도구와 워크플로우를 기반으로 한다.

사전 렌더링 및 디커플링의 핵심 원칙은 사이트와 애플리케이션이 이전보다 더 큰 자신감과 회복력을 가지고 전달될 수 있게 해준다.

# Get Started

VuePress (opens new window)

# yarn 설치
npm install -g yarn

# project folder 만들기
yarn create vuepress-site [project folder name]

# node package 설치
cd [project folder name]/docs
yarn

# 설정 수정 - /docs/src/.vuepress/config.js
# 테마 커스터마이징 - /docs/src/.vuepress/...

# 문서 만들기 - /docs/src/...
# 개발환경 실행
yarn dev

# 빌드하기
yarn build

# git repository 에 build push 하기
# https://vuepress.vuejs.org/guide/deploy.html#github-pages
# --------------------------
# file name: deploy.sh
# modified: 2021-01-26

#!/usr/bin/env sh

# abort on errors
set -e

# build
yarn build

# navigate into the build output directory
cd .vuepress/dist

# if you are deploying to a custom domain
# echo 'www.example.com' > CNAME

git init
git add -A
git commit -m 'deploy'

# if you are deploying to https://<USERNAME>.github.io
# git push -f git@github.com:<USERNAME>/<USERNAME>.github.io.git master

# if you are deploying to https://<USERNAME>.github.io/<REPO>
# git push -f git@github.com:<USERNAME>/<REPO>.git master:gh-pages

cd -
cd docs
# --------------------------

sh deploy.sh
